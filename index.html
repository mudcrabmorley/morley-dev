TECH

I use the following tools in our initial design, development and testing stages to ensure the delivery of a functional, user-friendly and well-crafted website that you absolutely love.

DESIGN TOOLS

WIRE FRAMING

I use apps such as Balsamiq to quickly sketch up layouts for your proposed website to give a basic idea of how it will look and function before development.

PROTOTYPING

If desired I can also give the design a more polished look using apps such as Figma to create a prototype of the website before fully developing it.

DEVELOPMENT TOOLS

CUSTOM CODING

I can create websites from scratch using HTML, CSS, and JavaScript to ensure they are lightweight and efficient. This also allows for more flexibility and keeps the cost down.

WORDPRESS

I can also create websites using WordPress. This allows you to easily update the content of your website yourself if desired. It is also a great way to integrate e-commerce functionality.

TESTING TOOLS

RESPONSIVENESS

Using the Chrome Developer Tools I can test the responsiveness of your website across all devices. This ensures that your website will look great on all screen sizes.

SPEED OPTIMISATION

Using tools such as Lighthouse gives an overview of how the website performs in terms of speed and accessibility. Techniques are then used to improve these metrics.
